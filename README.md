# Exploración de Componentes en Jetpack Compose

## Descripción

Este proyecto es una aplicación desarrollada en **Android Studio** utilizando **Jetpack Compose**, cuyo objetivo es demostrar el uso de diferentes componentes de interfaz (UI) modernos en Android.

Incluye ejemplos prácticos de contenedores, controles y elementos interactivos, permitiendo comprender su funcionamiento dentro de una aplicación real.


## Tecnologías utilizadas

*  Kotlin
*  Jetpack Compose
*  Material 3
*  Android Studio


## Componentes implementados

###  Contenedores

* LazyColumn
* LazyRow
* LazyVerticalGrid
* ConstraintLayout
* Scaffold
* Surface
* FlowRow
* FlowColumn
* BackdropScaffold

###  Controles e Interacción

* Card
* Checkbox
* Switch
* RadioButton
* Slider
* FloatingActionButton
* Icon
* Image
* ProgressBar

###  Navegación y UI adicional

* TopAppBar
* BottomNavigation (NavigationBar)
* NavigationRail
* TabRow
* Pager

###  Elementos auxiliares

* AlertDialog / Dialog
* Snackbar
* Divider
* Spacer
* DropdownMenu
* OutlinedTextField
* Tooltip
* Chip

##  Objetivo del proyecto

El objetivo principal es **explorar y aplicar los componentes más importantes de Jetpack Compose**, entendiendo su uso, personalización e integración en interfaces modernas.
